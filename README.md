# Exercise GIFs Backup

This repository serves as a backup host for exercise animation GIFs from the [Fitness Exercises with Animations](https://www.kaggle.com/datasets/edoardoba/fitness-exercises-with-animations/data) dataset on Kaggle.

The original GIF URLs in the dataset are no longer working, so this repo provides alternative hosting for those assets.
